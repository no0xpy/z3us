# Z3US
French: https://github.com/no0xpy/z3us/blob/main/README.md <br>
Z3US (Zeus) is a multitool coded in Python. It includes basic tools that can be useful in many areas. It is designed for Windows.<br><br>
<img src="https://forthebadge.com/badges/made-with-python.svg"></img>

# Overview
Here is the list of tools included in Z3US:<br>
 1 - IP Lookup<br>
 2 - Status Checker<br>
 3 - WHOIS Lookup<br>
 4 - Ports Scanner<br>
 5 - Identity Generator<br>
 6 - Phone Lookup<br>
 7 - Password Generator<br>
 8 - HTTP Requests Sender<br>
 9 - Discord Webhook Remover<br>
 10 - Multiple Searcher<br>
 11 - MiniFetch<br>

# Download
1 - Install Python: https://www.youtube.com/watch?v=7SSSocyUBnA (Please install the latest available version of Python) <br>
2 - Go back to the main project page and click on the latest available release in the "Releases" section <img src="https://github.com/no0xpy/z3us/blob/main/assets/releases.png?raw=true"></img><br>
3 - Click on "Source code (.ZIP)" which will download a file. Then, unzip this file.<br>
4 - Open a terminal in the folder and run:
```shell
pip install requirements.txt
```
<br>
5 - Double click on the "menu.py" file
